HTML introduction

what is HTML?

  • standart for Hyper text markup language
  • describes the structure of a web pages
  • consists of series of elements
  • elements tell the browser how to display the content
  • elemnts consist on pieces label whit content

explicacion de elementos basicos

  1. la etiqueta "DOCTYPE" define la declaracion del documento en cuestion como HTML
  2. la etiqueta HTML es el elemento principal de una pafina HTML
  3. la etiqueta "head" contiene informacion adicional acerca de la pagina HTML
  4. la etiqueta "title" especifica el titulo de la pagina HTML
  5. la etiqueta "body" especifica el cuetpo de la pagina web(todos los elementos visibles)
  6. la etiqueta "h1" define un texto de tamaño titular
  7. la etiqueta "h2" define un texto de tamaño subtitular
  8. la etiqueta "p" define un texto de tamaño parrafo

¿que es un elemento HTML?

un elemento HTML esta definido por una etiqueta de inicio, seguido de algun contenido y una etiqueta de finalizacion

  • <nombreEtiqueta>: etiqueta de inicio
  • contenido: entre la etiqueta de inicio y la etiqueta de finalizacion
  • </nombreEtiqeta>: etiqueta de finalizacion

Etiqueta <map>

la etiqueta <map> define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distintas areas detectables por el navegador, logrando identificar de manera independiente cada una de las areas.
la etiqueta <map> define las areas de la imagen mediante la etiqueta <area> con los atributos HTML "shape", que define la forma del area señalada en la imagen, "coords" que define las cooredenadas de cada uno de los puntos de la shape, "href" que puede definir un hipervinculo para cada una de las areas.

  • Ejemplo:

<img src="workflow.jpg" usemap="#workmap">
<map name="#workmap">
<area shape="rect" coords="34,44,270,350" href="pagina1.com">
<area shape="circle" coords="200,172,333,250" href="pagina2.com">
</map>

elemento Table en HTML

La etiqueta <table> permite colocar los datos en fila y columnas usando sus derivadas <tr> para definir una fila (table row), <th> para definir los encabezados de la tabla (table header) y <td> para asignar valores de las filas correspondientes a los encabezados de las columnas (table data)

  • Ejemplo:

<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
<th>Direccion</th>
</tr>

</tr<

Elementos HTML "DIV"

La etiqueta <div> es un elemnto que toma el ancho de la pagina por defecto y realiza automaticamente un salto de linea despues de cada elemnto. <div> se puede interpretar como contenedor de elementos HTML.

  • Ejemplo:
<div>
<h1>Titulo</h1>
<p>tecto interesante para el titulo</p>
</div>

elementos inline

son los elementos que no inician en una nueva linea de utilidad y unicamente toman el tamaño que requieren img, span, a

contents: permite algo asiendo que los elementos hijos

flex: muestra el contenedor o un contenedor flexible y vuelbe todos los elementos hijo en elementos inline

display grid: se utliza para crear un contenedor de cuadricula lo que activa en sus elementos hijo se le va a activar el modelo de el diseño
